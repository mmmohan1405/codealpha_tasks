Fitness Tracker

Overview

The Fitness Tracker is a console-based application developed in C that helps users record and monitor their daily fitness activities. The application allows users to log steps, calories burned, workout duration, view a fitness dashboard, and calculate Body Mass Index (BMI) through a simple menu-driven interface.

Features

- Log daily step count.
- Record calories burned.
- Track workout duration in minutes.
- View a daily fitness dashboard.
- Check daily step goal status.
- Calculate Body Mass Index (BMI).
- Display BMI category based on standard health ranges.
- Simple and user-friendly menu-driven interface.

Technologies Used

- C Programming Language
- Standard C Library
  - "stdio.h"

How to Compile and Run

Using GCC

Compile the program:

gcc fitness_tracker.c -o fitness_tracker

Run the executable:

./fitness_tracker

Using an Online Compiler

Copy and paste the source code into an online C compiler such as Programiz, OnlineGDB, or Replit, then run the program.
